The code in countdown_timer.txt was deployed to the homepage of Corporate Relocation site to provide a countdown timer for the corporate relocation. The code leverages Keith Wood's jquery plugin (http://keith-wood.name/countdown.html).

Build:
1) Add content editor webpart to a page
2) Edit content and switch to html view
3) add code in countdown_timer.txt to the html view
4) Close webpart and save page
